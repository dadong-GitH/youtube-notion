# 유튜브 → 노션 모니터링 파이프라인 설정 가이드

> 한국 기업 공식 유튜브 채널의 신규 영상을 매일 자동 수집하여 Notion 데이터베이스에 저장합니다.

---

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 모니터링 대상 | 한국 기업 공식 유튜브 채널 200개 (10개 산업군 × 20개 채널) |
| 수집 기준 | 조회수 100회 이상 |
| 자동 실행 | 매일 정오(12:00), macOS launchd |
| 저장 위치 | Notion 데이터베이스 |
| 뷰 구성 | 📋 전체 (테이블), 🏢 산업군별 (갤러리) |

---

## 디렉토리 구조

```
~/youtube-notion/
├── main.py               # 메인 수집 스크립트
├── channels.py           # 200개 채널 목록 정의
├── add_thumbnails.py     # 기존 페이지 썸네일 일괄 추가
├── run.sh                # launchd 실행 래퍼
├── .env                  # API 키 (비공개)
├── requirements.txt      # 패키지 목록
├── channel_cache.json    # 채널명 → channel_id 캐시
├── seen_videos.json      # 수집 완료 video_id 목록 (중복 방지)
├── search_cursor.json    # 미탐색 채널 순환 커서
└── monitor.log           # 실행 로그
```

### 2. .env 파일

```env
YOUTUBE_API_KEY=AIzaSy...
NOTION_TOKEN=ntn_...
NOTION_DATABASE_ID=...
```

### 3. Notion 연동

1. [notion.so/my-integrations](https://www.notion.so/my-integrations) → 인테그레이션 생성
2. 토큰 복사 → `.env`의 `NOTION_TOKEN`에 입력
3. 데이터베이스 페이지 우측 상단 `...` → `연결` → 인테그레이션 추가

## 실행 방법

### 수동 실행

```bash
cd ~/youtube-notion

# 일반 실행 (최근 1일, 공식채널만)
python3 main.py

# 검색 포함 (미탐색 채널도 브랜드명 검색)
python3 main.py --search

# 수집 기간 조정
python3 main.py --days 7

# 최초 1회: 전체 채널 ID 탐색
python3 main.py --setup

# 기존 페이지에 썸네일 일괄 추가
python3 add_thumbnails.py
```

### 자동 실행 (launchd)

macOS launchd를 사용하여 매일 정오(12:00)에 자동 실행됩니다.
잠자기 상태에서 전원이 켜지면 미실행된 작업이 자동으로 보상 실행됩니다.

**plist 위치:** `~/Library/LaunchAgents/com.youtube-notion.monitor.plist`

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.youtube-notion.monitor</string>
    <key>ProgramArguments</key>
    <array>
        <string>/bin/bash</string>
        <string>/Users/dadong/youtube-notion/run.sh</string>
    </array>
    <key>StartCalendarInterval</key>
    <dict>
        <key>Hour</key>
        <integer>12</integer>
        <key>Minute</key>
        <integer>0</integer>
    </dict>
    <key>StandardOutPath</key>
    <string>/Users/dadong/youtube-notion/monitor.log</string>
    <key>StandardErrorPath</key>
    <string>/Users/dadong/youtube-notion/monitor.log</string>
    <key>RunAtLoad</key>
    <false/>
</dict>
</plist>
```

**launchd 관리 명령어**

```bash
# 등록 (최초 1회)
launchctl load ~/Library/LaunchAgents/com.youtube-notion.monitor.plist

# 해제
launchctl unload ~/Library/LaunchAgents/com.youtube-notion.monitor.plist

# 즉시 실행 (테스트)
launchctl start com.youtube-notion.monitor

# 로그 확인
tail -f ~/youtube-notion/monitor.log
```

## YouTube API 할당량

| 메서드 | 단위 | 사용 시점 |
|--------|------|----------|
| `playlistItems.list` | 1 unit/채널 | 공식채널 일일 수집 |
| `search.list` | 100 units/채널 | 미탐색 채널 검색 (`--search`) |
| `videos.list` | 1 unit/50개 | 영상 상세 정보 조회 |
| `channels.list` (handle) | 1 unit | `--setup` 채널 ID 탐색 |

- **일일 한도:** 10,000 units (PDT 자정 00:00 / KST 16:00 리셋)
- 공식채널 97개 × 1 = 약 97 units/일 (여유 충분)
- `--search` 추가 시 최대 90개 채널 × 100 = 9,000 units (주의)

## Notion 뷰 구성

### 📋 전체 (테이블 뷰)
- 전체 영상 목록
- 정렬: 업로드 날짜 내림차순

### 🏢 산업군별 (갤러리 뷰)
- 산업군별 그룹화
- 카드 미리보기: 페이지 커버 (유튜브 썸네일)
- 갤러리 뷰에서 썸네일 이미지 확인 가능

---

## 갤러리 썸네일 설정

영상 저장 시 유튜브 썸네일이 페이지 커버로 자동 설정됩니다.

```python
# thumbnail URL 패턴
f"https://img.youtube.com/vi/{video_id}/hqdefault.jpg"

# Notion 페이지 커버 설정
cover={"type": "external", "external": {"url": thumbnail_url}}
```

기존 페이지에 썸네일 일괄 추가:
```bash
python3 add_thumbnails.py
```

## 중복 방지 로직

- `seen_videos.json`에 수집된 `video_id` 저장
- 수집 전 해당 ID 포함 여부 확인 → 중복 스킵
- 채널 ID는 `channel_cache.json`에 캐시 (반복 API 호출 방지)

## 문제 해결

### API 할당량 초과
```
HttpError 403: quotaExceeded
```
→ YouTube API 할당량 소진. KST 16:00 이후 자동 리셋.

### Notion 401 오류
→ `.env`의 `NOTION_TOKEN` 확인, 인테그레이션이 해당 데이터베이스에 연결됐는지 확인.

### Notion 404 오류
→ `NOTION_DATABASE_ID` 확인, 데이터베이스 페이지에 인테그레이션 연결 필요.

### launchd 미실행
```bash
launchctl list | grep youtube-notion
```
→ 출력 없으면 plist 재등록 필요.
